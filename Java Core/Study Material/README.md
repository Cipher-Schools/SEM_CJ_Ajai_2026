---

## 💡 Tips for Students

- **Type out every example** — don't just read. Muscle memory matters in programming.
- **Modify examples** — change values, remove things, break them. Understanding why something breaks = deep learning.
- **Attempt practice questions** before looking anything up.
- **Compile and run everything** — if you see a red underline or compile error, fix it yourself first.
